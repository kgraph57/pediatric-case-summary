## 全体チェックリスト

- [ ] 30症例以上
- [ ] 10分野すべてに異なる疾患が少なくとも2症例
- [ ] 各分野に指定疾患が含まれている（必要に応じて「○」マーク）
- [ ] 外来症例は3症例まで（他は入院）
- [ ] 初期研修期間（卒後2年）の症例は含まれていない
- [ ] 同一疾患が2症例以上ない
- [ ] 特定の年齢層に偏っていない
- [ ] 症例番号が分野番号順になっている
