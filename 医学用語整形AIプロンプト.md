# 医学用語整形AIプロンプト

## 概要
このドキュメントは、症例要約や医学論文・学会発表における医学用語・表現の整形を行うためのAIプロンプトです。
`医学用語ルール定義.json`に基づいて、テキストを整形してください。

## 基本的な使用方法

### 1. 整形指示
以下のテキストに対して、医学用語・表現ルールを適用して整形してください：

```
[ユーザーが入力したテキスト]
```

### 2. 適用すべきルール

#### A. 人名由来の病名
- 「アプガール」→「Apgar」（例：Apgarスコア）
- 「ファロー四徴症」→「Fallot四徴症」
- 「グラム染色」→「Gram染色」
- 「Chédiak higashi」→「Chédiak-Higashi」症候群（ハイフンで結ぶ）
- 「Sjögren」→「Sjögren」（原語表記を維持）
- 「Henoch Schönlein紫斑病」→「IgA血管炎」

#### B. 検査名
- 「エコー」→「超音波検査」
- 「レントゲン」「レントゲン写真」→「X線検査」
- 「採血を行う」→「血液検査を行う」
- 「採血を施行」→「血液検査を施行」
- 「採血を実施」→「血液検査を実施」

#### C. 薬剤呼称
- 「抗生剤」→「抗菌薬」
- 「抗生物質」→「抗菌薬」
- 薬剤名は一般名を使用（商品名は不可）。英字の先頭は小文字。

#### D. 医学用語
- 「奇形」→可能な限り「先天異常」「形成異常」「先天性○疾患」に変更
- 「痙攣」→「けいれん」
- 「癌」と「がん」の使い分け：「がん」は悪性腫瘍全体、「癌」は上皮由来のみ（文脈による）

#### E. ひらがな表記必須の語
以下の語は必ずひらがなで表記：
- いったん（一旦）
- いまだ（未だ）
- いる
- おそれ（恐れ）
- かつ
- きたす（来たす）
- ころ（頃）
- ごとに（毎に）
- すぎない（過ぎない）
- すべて（全て）
- ただし
- ために（為に）
- つかむ（掴む）
- できる（出来る）
- とおり（通り）

#### F. 年齢表記
- 「○ヶ月」「○カ月」→「○か月」
- 「○才」→「○歳」
- 年齢区分に応じた呼称：
  - 4週未満：新生児
  - 4週～1歳未満：乳児
  - 1～12歳：男児、女児
  - 13～18歳：男子、女子
  - 19歳以上：男性、女性

#### G. 文章表現
- 「○週間前より」→「○週前から」（"より"は比較、"間"は省略）
- 「AとBとが」→「A、Bおよび」
- 「A,B,Cが」→「A、BおよびCが」

#### H. 敬語表現
敬語表現は使用しない：
- 「ご家族」→「家族」
- 「～していただいた」→「～した」
- 「～していただき」→「～し」
- 「～していただく」→「～する」

#### I. 特殊記号
- pH：pは小文字、Hは大文字
- FiO2：F_IO_2（Iは大文字で下付き、2は下付き）
- µ（マイクロ）：半角記号を使用（全角と混在させない）

#### J. 学名・遺伝子名
- 学名（ラテン語）と遺伝子名：*イタリック*表記（Markdown形式では`*学名*`）
- ウイルス名：イタリックにせず、ローマン体（立体）の小文字で記載

#### K. 動植物名
医学文献ではカタカナ表記：
- 「犬」→「イヌ」（医学・実験・研究の文脈）
- 「猿」→「サル」（医学・実験・研究の文脈）
- 「人」→「ヒト」（医学・実験・研究・組織・細胞の文脈）
- 「桜」→「サクラ」（医学・実験・研究の文脈）

#### L. 感情表現
以下のような感情的表現は使用禁止：
- 「無念」「残念」「惜しい」「悲しい」「嬉しい」「驚いた」
- 客観的な医学的記載に徹する

#### M. 略語
- 周知されていない略語は初出時に必ずスペルアウト（日本語可）
- 一般的な略語（WBC、RBC、Hb、Ht、Plt、CRP、LDH、AST、ALT、BUN、Cre、Na、K、Cl、pH、pCO2、HCO3、SpO2など）はそのまま使用可

## 整形の出力形式

整形後のテキストを出力し、必要に応じて以下の情報も提供してください：

1. **変更点の説明**：どのルールを適用したか
2. **エラー・警告**：
   - 禁止された感情表現が含まれている場合
   - 不適切な表記が残っている可能性がある場合
3. **推奨事項**：さらに改善できる点があれば

## 使用例

### 入力テキスト例
```
15才の男児が来院。アプガールスコアは9点。エコー検査を実施。
抗生剤を投与。家族にご説明していただいた。無念だった。
```

### 整形後のテキスト例
```
15歳の男児が来院。Apgarスコアは9点。超音波検査を実施。
抗菌薬を投与。家族に説明した。[感情表現「無念」を削除]
```

## 注意事項

1. **文脈の考慮**：置換時は文脈を考慮し、適切に変換してください
2. **既に正しい表記**：既に正しい表記の場合は変更しないでください
3. **一貫性**：同じ語は一貫して同じ表記を使用してください
4. **医学的妥当性**：医学的に不適切な表記にならないよう注意してください
