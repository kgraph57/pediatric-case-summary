# 症例要約作成ルール

## 1. 症例要約（前半）：基本情報・ヘッダー

### 基本入力ルール

- **症例番号**：指定疾患の症例には、番号の前に「○」を記載する。

- **入院・外来**：該当する方にチェックを入れる（チェックしなかった項目は消す）。

    - ※外来症例は3症例まで。

- **受け持ち期間**：ローテ期間内であるように記載する。

- **診断名**：人名由来の病名は英語表記とする（例：Down症候群）。

- **特記事項**：家族歴、妊娠・分娩歴、既往歴は、疾患に関係あるものを記載する。

### 年齢の記載ルール（受け持ち開始時点）

- **生後0日～1か月まで**：日齢〇日

- **生後1か月～11か月**：〇か月

- **生後1歳0か月～1歳11か月**：1歳〇か月

- **生後2歳0か月以降**：〇歳

---

## 2. 症例要約（後半）：本文・記述フォーマット

### 全体設定

- **分量**：症例要約だけで**30行以内**。

- **文字サイズ**：10.5pt ～ 12pt。

- **改行**：各項目（【主訴】【現病歴】など）の冒頭でのみ改行し、項目内では改行しない。

### 文体・表現

- **句読点**：句読点（ 、。）で統一する。カンマ（，）やピリオド（．）は不可。

- **敬語**：「ご家族」「～していただいた」などの敬語表現は不要。「～した」等で統一する。

- **略語**：正式名称で記載する。一度スペルアウトすれば以降は略語使用可。

### 数字・単位・記号

- **数字**：すべて半角表記。

- **単位のスペース**：

    - **原則**：単位の前には半角スペースを入れる（例：`Na 135 mEq/L`）。

    - **例外（スペースなし）**：「点」「日」「g（グラム）」「度」「回/分」「秒」「万/µL」「mm」などの前はスペース不要（例：`5点` `日齢2日` `1871g` `37.7度` `30回/分` `101.7秒` `52.2万/µL` `3.6mm`）。

    - **WBCの特例**：WBCのみ数字と単位の間のスペースなし（例：`WBC15580/μL`）。

- **特殊記号**：

    - **μ（マイクロ）**：全角ではなく半角記号を使用。

    - **L（リットル）**：大文字を使用（例：`mg/dL`）。

---

## 3. その他の注意点・特殊表記

### 記載内容の注意

- **特記事項の精査**：家族歴・妊娠分娩歴・既往歴は、画一的に「特記すべきことなし」とせず、疾病に関係のあるものを記入する。

- **フォーマット統一**：コピペを疑われないよう、数字の全角半角混在、フォントの不統一、文末形式の不統一がないよう徹底する。

### 特殊な医学用語・記号

- **FiO2**：「F」の後の「I」は大文字で下付き、「2」は半角下付きとする（$F_IO_2$ 表記）。

- **ウイルス名**：イタリックにせず、ローマン体（立体）の小文字で記載。

- **遺伝子名**：*イタリック（斜体）*で記載。

- **括弧（ ）の用法**：

    - 半角スペース ＋ 半角括弧開く ＋ 内容 ＋ 半角括弧閉じる ＋ 半角スペース

    - 例：`テキスト ( 内容 ) テキスト`

## 5. 症例の選び方・記載ルール

### 基本ルール

* 実在しており、実際に担当した症例であること（架空は不可）

* 担当チームに氏名があれば、主治医でなくても、途中で交代しても構わない

* 診療録（カルテ）に受験者が診療内容について記載していること

* 後輩と同じ症例でも一緒に診た症例なら適法だが、**別々の書き方をすること（コピペ厳禁）**

* 各分野の疾患は一覧表に従い、指定疾患に注意する

* **初期研修期間の症例（卒後2年）は不可**

* 10の各分野に異なる疾患が少なくとも2症例必要

* 外来症例は3症例まで（他は入院患者）

* 形式・ルールを守り、様式を統一し、細部まで目を光らせる

### 症例要約の必須記載事項（各2点、10点満点で評価）

1. **要約の簡潔さ**

2. **診断のアプローチ、臨床判断**

    * 【鑑別診断】は必須。【症例の問題点】も記載。

    * *NG例*: 染色体の項目（Down症候群＋肺炎）で、鑑別・説明も含めて肺炎ばかり記載するのは大きく減点または受験不可

3. **治療の適切さ**

4. **インフォームドコンセント（倫理的配慮）**

    * 記載例：『以上家族へ説明し同意を得た』 or 『以上説明し同意を得て治療開始した』

    * *NG例*: 「説明しただけ」は不可

    * **段落の最後に書くと採点上有利**

5. **転帰と退院後の具体的な指導（患者および家族）**

    * 【家族への説明】【退院後の経過】

## 6. 減点項目・禁止事項チェックリスト

### コピペ対策・フォーマット

* **厳禁**: コピペを疑われること。

* **NG**: 異なる字体（ゴシック、明朝）の混在、文字サイズがバラバラ。

* **NG**: 全角と半角の混在、mlとmLの混在。

### 減点・受験不可となる項目

以下に該当すると「受験不可」または「大幅減点」となります。

* [ ] 30例未満

* [ ] 指定疾患が0例の分野がある

* [ ] 各分野の症例が0例または1例

* [ ] 同一疾患が2症例ある

* [ ] 第1病名が要約と一覧表で不一致

* [ ] 特定の年齢層（新生児など）に偏っている

* [ ] 症例番号が分野番号順ではない

* [ ] 指定疾患に「レ」がない

* [ ] 不適当な分野の選択

* [ ] 記載漏れ（性別・転帰等）

## 7. 小児科専門医試験のお約束（表記ルール）

### 1. 字体・人名・用語

* **字体と句読点の統一**: 「ゴシックと明朝」「。と、」「、と，」を混在させない（コピペ疑い回避）

* **人名は原語**: 二人のときはハイフンで結ぶ

    * ×アプガール → ○ **Apgar** スコア

    * ×ファロー四徴症 → ○ **Fallot** 四徴症

    * ○ **Gram** 染色

    * ×Chédiak higashi → ○ **Chédiak-Higashi** 症候群

    * ○ **Sjögren** 症候群

    * ×Henoch Schönlein 紫斑病 → ○ **IgA血管炎**

* **検査表記**: 「○検査」とする

    * ×エコー → ○ **超音波検査**

    * ×レントゲン写真 → ○ **X線検査**

    * ×採血を行う → ○ **血液検査を行う**

* **動植物名**: カタカナ表記（イヌ、サル、ヒト、サクラなど）

* **感情表現**: 「無念」などの感情的表現は不可

### 2. 年齢・月齢の表記

* **区分ごとの呼称**:

    * **4週未満**: 新生児（性別に特別な意味がある場合のみ、男児/女児）

    * **4週～1歳未満**: 乳児（性別に特別な意味がある場合のみ、男児/女児）

    * **1～12歳**: 男児、女児

    * **13～18歳**: 男子、女子

        * *NG例*: 15才男子 → ○ **15歳の男子**

    * **19歳以上**: 男性、女性

* **記載形式**:

    * 1か月児まで：**生後日数**

    * 1歳児まで：**月数**

    * 2歳児まで：「**1歳何か月**」

    * ×6ヶ月、6カ月 → ○ **6か月**

### 3. 文章表現

* **列挙**: A、BおよびCが（×AとBとが。×A,B,Cが）

* **時間経過**: ×～週間前より → ○ **～週前から**（"より"は比較、"間"は省略）

## 8. 【常識】日本語記載の原則（症例要約・論文共通）

### 書式・スペース

* **段落の文頭**: 全角空ける。

* **半角括弧**: 前後は必ず半角空ける。

* **半角記号（, - : ;）**: **後**は必ず半角空ける。（**前**には空けない）。

* **数値と単位の間**: 半角空ける（例: `10 kg`, `0.1 mg`, `mL`）。

    * *例外*: `%`, `°C` は通常空けない。「万」の前は空けない。

    * **マイクロ（µ）**: 半角（英語字体）を使用。全角と混在させない。

### 文法・語法

* **主語の統一**: 「医者は」または「患者は」。常に意識して統一する。

* **体言止めの禁止**: 下品であるため、できるだけ文章にする。

* **助詞抜き禁止**:

    * ×生後心房中隔欠損症指摘 → ○生後、心房中隔欠損症**と**指摘された。

    * ×～診断で加療目的入院 → ○～**と**診断されて加療目的**に**入院した。

### 専門用語の表記ルール

* **略語**: 周知されていない略語は初出時に必ずスペルアウト（日本語可）。

* **薬物名**: 一般名を使用（商品名不可）。英字の先頭は小文字。

* **薬剤呼称**: 「～薬」とする（×抗生剤 → ○抗菌薬、利尿薬、強心薬）。

* **学名（ラテン語）**: *イタリック*を使用（例: *Escherichia coli*）。

* **遺伝子名**: *イタリック*を使用。

* **「がん」の使い分け**: 「がん」は悪性腫瘍全体、「癌」は上皮由来のみ。

* **ひらがな表記する語**: いったん、いまだ、いる、おそれ、かつ、きたす、ころ、ごとに、すぎない、すべて、ただし、ために、つかむ、できる、とおり。

### 特殊記号・化学式

* **下付き文字**: $NH_3$, $SpO_2$

* **イオン**: $HCO_3^-$（3は下付き、マイナスは上付き）

* **FiO2**: $F_IO_2$（Iは大文字で下付き、2は下付き）

* **pH**: pは小文字、Hは大文字。

---

## 9. その他用語・記載順の注意

* **奇形**: 「先天異常」「形成異常」「先天性○疾患」を推奨。症状は「変質徴候」。

* **痙攣**: 「けいれん」とひらがな表記。

* **現症の記載順**: 身長 → 体重 → 体温 → 呼吸数 → 脈拍 → 血圧

* **検査の記載順**: 血液 → 血清生化学 → 尿

* **静脈血液ガス**: 特異的な結果がない場合、pHやpCO2は不要。Lactateは代謝疾患以外では不要。
