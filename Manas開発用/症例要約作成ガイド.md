# 症例要約作成ガイド：ステップバイステップ

## 📋 全体の流れ

1. **症例の選定と整理**（事前準備）
2. **情報収集**（各症例ごと）
3. **症例要約の作成**（各症例ごと）
4. **チェックと修正**（各症例ごと）
5. **全体の最終確認**（全症例完成後）

---

## 🎯 Phase 1: 症例の選定と整理（事前準備）

### TODOリスト

  - 初期研修期間（卒後2年）の症例は除外

## 📝 Phase 2: 情報収集（各症例ごと）

### ステップバイステップ

#### Step 1: カルテ・診療録の確認
1. 該当症例のカルテを開く
2. 診療録に自分が記載した内容があるか確認
3. 受け持ち期間を確認

#### Step 2: 基本情報の収集
1. **患者ID**を確認
2. **受け持ち期間**を確認（ローテ期間内であること）
3. **受け持ち開始時点の年齢**を確認（年齢表記ルールに従う）
4. **性別**を確認
5. **転帰**を確認（治癒・軽快・不変・増悪・死亡）

#### Step 3: 病歴情報の収集
1. **周産期歴**（疾患に関係あるもののみ）
   - 在胎週数、出生体重、分娩方法
   - Apgarスコア（必要な場合）
   - 母体情報（必要な場合）：年齢、経産歴、母体感染症など

2. **既往歴**（疾患に関係あるもののみ）
   - 新生児マススクリーニング
   - その他の既往歴

3. **家族歴**（疾患に関係あるもののみ）

4. **感染接触歴**（必要な場合）
   - 家族・保育園・学校での感染症の流行
   - 生ものの摂食歴

5. **生活歴**（必要な場合）
   - 離乳食、栄養方法など

#### Step 4: 診断名の確認
1. 第1病名を確認
2. 第2病名以降を確認（必要に応じて）
3. 指定疾患かどうか確認（指定疾患には「○」マーク）

#### Step 5: 臨床情報の収集
1. **主訴**を確認
2. **現病歴**を時系列で整理
3. **入院時（来院時）診察所見**
   - バイタルサイン
   - 身体所見（記載順：身長 → 体重 → 頭囲 → 体温 → 呼吸数 → 脈拍 → 血圧）

#### Step 6: 検査データの収集
1. **血液検査**
   - WBC、Hb、Hct、Plt
   - 血清生化学（T.Bil、AST、ALT、LDH、BUN、Cre、Na、K、Cl、CRPなど）
   - 凝固系（PT-INR、APTT、Fibrinogen、D-dimer、PIVKA-2など）

2. **静脈血液ガス**（特異的な結果がある場合のみ）

3. **特殊検査**（低血糖時など、必要な場合）
   - インスリン、アセト酢酸、3-ヒドロキシ酪酸、遊離脂肪酸など

4. **尿検査**（必要な場合）
   - カテーテル尿検査、中間尿検査
   - 比重、白血球反応、亜硝酸塩、白血球、尿Gram染色など

5. **その他の検査**
   - 超音波検査、X線検査、CT、MRI、心電図検査など

#### Step 7: 経過情報の収集
1. **鑑別診断**を整理（必須）
2. **症例の問題点**を整理（記載推奨）
3. **入院後（来院後）経過**を時系列で整理
4. **家族への説明・指示**の内容を確認
5. **退院後の経過**を確認（記載推奨）

## ✍️ Phase 3: 症例要約の作成（各症例ごと）

### ステップバイステップ

#### Step 1: テンプレートに情報を入力
1. `症例情報収集テンプレート.md`を開く
2. 各項目に収集した情報を入力
3. 情報が不足している場合は、再度カルテを確認

#### Step 2: 情報を私に共有
1. テンプレートに入力した内容をコピー
2. または、カルテから抜き出した情報をそのまま共有
3. 症例番号と分野番号を明記

#### Step 3: 私が症例要約を作成
1. 私がルールに従って症例要約を作成
2. 30行以内に収める
3. 各項目の冒頭でのみ改行
4. 表記ルールを遵守

#### Step 4: 作成された要約を確認
1. 内容が正確か確認
2. 不足している情報がないか確認
3. 修正が必要な点があれば指摘

#### Step 5: 修正と調整
1. 私が修正を反映
2. 必要に応じて繰り返し調整

## ✅ Phase 4: チェックと修正（各症例ごと）

### チェックリスト

#### 基本情報のチェック
- [ ] 症例番号が正しい
- [ ] 分野番号が正しい
- [ ] 患者IDが正しい
- [ ] 受け持ち期間がローテ期間内
- [ ] 年齢表記がルールに従っている
- [ ] 性別が正しい
- [ ] 転帰が正しい
- [ ] 入院・外来の区分が正しい

#### 診断名のチェック
- [ ] 第1病名が要約と一覧表で一致
- [ ] 指定疾患に「○」マークがある（必要な場合）
- [ ] 人名由来の病名は英語表記

#### 記載内容のチェック
- [ ] 【鑑別診断】が記載されている（必須）
- [ ] 【症例の問題点】が記載されている（推奨）
- [ ] 【家族への説明・指示】が記載されている（必須）
- [ ] インフォームドコンセントの記載がある（「説明し同意を得た」など）
- [ ] 退院後の具体的な指導が記載されている

#### 表記ルールのチェック
- [ ] 数字はすべて半角
- [ ] 単位のスペースが正しい
  - 原則：半角スペース（例：`12.4 g/dL`）
  - 例外：点、日、g、度、回/分、秒、万/µL、mm、%、mg/kg/dayはスペースなし
- [ ] WBCの表記が正しい（`WBC 13560/μL`のように、WBCと数字の間にスペース）
- [ ] 句読点は、。で統一（カンマ、ピリオドは不可）
- [ ] 敬語を使用していない（「～した」で統一）
- [ ] 略語は初出時にスペルアウト（以降は略語可）
- [ ] 検査名は「○検査」形式（×エコー → ○超音波検査）
- [ ] µ（マイクロ）は半角記号
- [ ] L（リットル）は大文字

#### フォーマットのチェック
- [ ] 各項目（【主訴】など）の冒頭でのみ改行
- [ ] 項目内では改行していない
- [ ] 30行以内に収まっている
- [ ] 文字サイズは10.5pt～12pt
- [ ] フォントが統一されている（ゴシックと明朝の混在なし）

#### 内容のチェック
- [ ] 要約が簡潔である
- [ ] 診断のアプローチ、臨床判断が記載されている
- [ ] 治療の適切さが記載されている
- [ ] 転帰と退院後の具体的な指導が記載されている
- [ ] 特記事項（家族歴・妊娠分娩歴・既往歴）は疾患に関係あるもののみ記載

## 🔍 Phase 5: 全体の最終確認（全症例完成後）

### 最終チェックリスト

#### 基本要件の確認
- [ ] 30例以上ある
- [ ] 指定疾患が0例の分野がない
- [ ] 各分野の症例が0例または1例でない
- [ ] 同一疾患が2症例以上ない
- [ ] 外来症例は3症例まで
- [ ] 特定の年齢層に偏っていない
- [ ] 症例番号が分野番号順になっている
- [ ] 指定疾患に「レ」がない（必要な場合）

#### 一覧表との整合性確認
- [ ] 第1病名が要約と一覧表で一致している
- [ ] 症例番号が正しく振られている
- [ ] 分野番号が正しい

#### フォーマット統一の確認
- [ ] 数字の全角半角混在がない
- [ ] フォントの不統一がない
- [ ] 文末形式の不統一がない
- [ ] コピペを疑われないよう、細部まで統一されている

#### 記載漏れの確認
- [ ] 性別が記載されている
- [ ] 転帰が記載されている
- [ ] 必須項目がすべて記載されている

## 📌 効率的な作業のコツ

### 1. バッチ処理
- 同じ分野の症例をまとめて作成すると効率的
- 類似した症例は、表記を統一しやすい

### 2. チェックリストの活用
- 各症例作成時にチェックリストを使用
- 後で修正する手間を減らす

### 3. 段階的な進捗管理
- まず10症例を作成して確認
- 問題がなければ残りを進める

### 4. 情報の整理
- カルテから情報を抜き出す際は、テンプレートに直接入力
- 後で見返しやすいように整理

### 5. 私との連携
- 情報をまとめて共有すると効率的
- 不明な点は早めに確認
